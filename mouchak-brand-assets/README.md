## Logo Mark (9 Hexagons - Solid Gold)

The Mouchak logo represents a beehive — 9 connected hexagons forming a diamond/arrow shape pointing upward, symbolizing coordination, growth, and collective intelligence.

### Colors

| Name | Hex | Usage |

| **Mouchak Brown** | `#92400E` | Stroke/outline |

| **Darker BG** | `#111827` | Darkest backgrounds |

## Typography

- **Primary Font**: System UI / -apple-system / sans-serif
- **Logo Text**: 600 weight (semibold)
- **Bengali Text**: মৌচাক (for cultural identity)

---

## Brand Guidelines

1. **Minimum Size**: Logo mark should not be smaller than 16×16px
2. **Clear Space**: Maintain padding of at least 1 hexagon width around the logo
3. **Background**: Use gold on dark backgrounds, darker gold on light backgrounds
4. **Don't**: Rotate, stretch, add effects, or change colors
